# 🌄 Explore Coimbatore

A responsive tourism website showcasing popular attractions across different regions of **Coimbatore**.

> **Note:** This project was created as part of my learning journey using extensive AI assistance. I used AI tools to understand web development concepts, generate code, and learn how a multi-page website is structured. The project represents my early exploration of HTML and CSS rather than independent professional web development.

---

## 📖 Overview

Explore Coimbatore is a static website that presents tourist attractions from different regions of Coimbatore through a clean and organized interface. The project helped me become familiar with the basics of web page structure, styling, navigation, and responsive layouts.

---

## ✨ Features

- 🏞️ Explore tourist attractions across Coimbatore
- 🧭 Region-wise navigation
- 📄 Individual pages for popular destinations
- 📱 Responsive layout
- 🎨 Clean user interface
- 🖼️ Image-based attraction cards

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Responsive Web Design

---

## 🚀 Running the Project

Clone the repository:

```bash
git clone https://github.com/Thaarani1/EXPLORE-COIMBATORE.git
```

Open `index.html` in any modern web browser.
